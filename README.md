# 📊 SQL Data Analysis Project

## 📌 Project Overview

This project was completed as part of the **DecodeLabs Data Analytics Internship – Project 3**. The objective is to analyze an e-commerce dataset using SQL queries to extract meaningful business insights. The project demonstrates fundamental SQL operations such as data filtering, sorting, grouping, aggregation, and reporting.

---

## 🎯 Objectives

* Retrieve data using SQL queries
* Filter records using `WHERE`
* Sort data using `ORDER BY`
* Group data using `GROUP BY`
* Perform aggregations using `COUNT`, `SUM`, and `AVG`
* Generate business insights from transactional data

4. Run all notebook cells sequentially.
